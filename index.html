Instalação

Primeiro Passo

Tudo começa com o download da versão cliente, na tabela acima você encontra os downloads de todas as versões. Após o download, extraia o arquivo .zip para uma pasta no local de sua preferência.


Segundo Passo


Abra a pasta que estão os arquivos e abra o executavel "MultiMc.exe", selecione a linguaguem que preferir e também confirme a versão do Java a ser utilizado. (preferivel Java 1.8 - x64).

Na mesma aba, na parte de baixo, tem a opção para alocar mais memória para o minecraft, e para o modpack isso é essencial. Na tabela abaixo é possivel ver as opções de memória necessarias para jogar o modpack. (não foi testado com menos de 4gb de memória)

Você deve alterar a memória em "MAXIMUM MEMORY ALLOCATION", "MEMÓRIA MÁXIMA", se seu programa estiver em português.

Abaixo está uma tabela de valores para a quantidade de memória.

RAM total RAM para o Modpack RAM em MB
4 GB n/a n/a
8 GB 4 á 6 GB 4.096 - 6.144 MB
16 GB 6 á 8 GB 6.144 - 8.196 MB
32 GB 6 á 8 GB 6.144 - 8.196 MB

Terceiro Passo


Abra o modpack.


Possíveis configurações para melhor performance

O modpack conta com algumas configurações para melhora de performance, cada computador responde de um jeito. Você pode tentar remover o mod Optifine, em alguns computadores ele causa mais lag do que ajuda. Junto com alguns mods existe o BetterFPS, que as configurações dele é no menu do minecraft e é possivel alterar o algoritimo de execução, teste alguns e veja qual é o melhor para você.
